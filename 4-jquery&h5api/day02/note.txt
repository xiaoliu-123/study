2.attr方法和prop方法的区别  
    对于HTML元素本身就带有的固有属性，在处理时，使用prop方法。
    对于HTML元素我们自己自定义的DOM属性，在处理时，使用attr方法。
    具有 true 和 false 两个属性的属性，如 checked, selected 或者 disabled 使用prop()

    发送请求给后台
    如果成功，则提示用户保存成功

        $.ajax({
            id: user ? user.id : '',    判断是新增的还是修改的
            url:'',
            method:'',
            data:{},
            success:function(res){
                //提示用户保存成功

                    alert('保存成功');

                }
            },

修改用户信息

    修改按钮用事件代理绑定click事件，
      首先获取当前要修改的用户对象 user = $(this).attr('data-item');
      解析该对象 user = JSON.parse(user); （因为前面拿的是JSON格式的）
      再把对象数据设置到表单里面 
      之后设置模态框标题   $('.dialog-header').html('修改用户信息');
      最后显示模态框  $('.dialog').fadeIn();
